# SPYTube

SPYTube is a streaming application for Android that provides access to movies, TV series, and anime. It features a transparent glass interface and a custom video player designed for mobile devices.

## Features

*   **Glass Interface**: The UI uses a transparent design with blur effects.
*   **Video Player**: Includes gesture controls for volume and brightness, plus a long-press feature for 2x playback speed.
*   **Content Library**: Access to movies, television series, and anime.
*   **Watch History**: Automatically saves playback progress.
*   **Favorites List**: Allows users to save content for later.
*   **Immersive Mode**: Hides system bars for full-screen viewing.

## Installation

1.  Download the **SPYTube.apk** file from this repository.
2.  Open the file on your Android device.
3.  If prompted, allow installation from unknown sources.
4.  Launch the application.

## Requirements

*   Android 8.0 (Oreo) or higher.
*   Active internet connection.
