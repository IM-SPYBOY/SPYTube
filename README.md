## What is SPYTube?

SPYTube is a free Android application that brings together movies, TV series, and anime into one clean interface. It uses a frosted glass design language and ships with a custom video player that supports gesture-based controls.

No sign-ups. No subscriptions. Just open it and watch.

---

## Highlights

| Feature | Description |
|---|---|
| **Glass UI** | Transparent panels with real-time blur effects across the entire interface. |
| **Gesture Controls** | Swipe for volume and brightness. Long press for 2x speed. No buttons needed. |
| **Watch History** | Picks up where you left off, automatically. |
| **Favorites** | Save anything to your personal list for later. |
| **Full-screen Player** | Immersive mode hides all system bars, edge to edge. |
| **Search** | Find content instantly with metadata pulled from TMDB. |

## Download

Grab the latest version from the **[Releases](https://github.com/IM-SPYBOY/spytube/releases)** page.

**How to install:**
1. Download `SPYTube.apk` from the latest release.
2. Open the file on your Android device.
3. Allow installation from unknown sources if prompted.
4. Done.

**Requires:** Android 8.0 or higher.
